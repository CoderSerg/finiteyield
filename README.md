# Finite Yield

The best **FORKED** command line script for roblox.

# Features FY adds
### Quotes
For some reason, base IY doesn't have this, so I added it! I don't really know what commands could use this other than `notify` but now you can use double or single quotes.

### Lucide Icons
FY replaces legacy icons that IY uses with [Lucide Icons](https://lucide.dev), which I think look way better than the regular icons, and it also makes the icons less blurry, so that's pretty good.

**Icons are used from [icons.rest](https://icons.rest)*

### Theme presets
FY adds theme presets as a setting with themes like: 
```
Catppuccin Mocha
Dracula
Nord
Tokyo Night
One Dark
Rose Pine
Gruvbox
Classic
```
**Light themes coming soon*

### Bug fixes and small improvements
FY fixes most bugs I found while testing IY, like the menu closing if you are focused on the command bar while opening settings, and I made some improvements to things like notifications.

## Usage
Use the reference menu in the script to see full features, which is next to the settings icon as an info icon.
You can learn how to use most of the features of this script in [the wiki](https://github.com/EdgeIY/infiniteyield/wiki)!
